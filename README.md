# AIT-620: Methods of AI and ML – Week 7 Assignments

This repository contains the **Week 7 assignments** for the course **AIT-620: Methods of AI and Machine Learning**. The assignments demonstrate data preprocessing, analysis, and machine learning techniques using Python and Jupyter Notebooks.

## Contents

### 1. Assignment 7.1 – Data Preprocessing
- **File:** `Assignment 7.1.ipynb`  
- **Description:** This notebook focuses on cleaning and preparing the Canada Immigration dataset.  
- **Key Steps:**
  - Removal of unnecessary administrative columns
  - Renaming columns for clarity
  - Standardization of column labels
  - Aggregation of yearly immigration data
  - Calculation of total immigration per country
- **Outcome:** A clean, analysis-ready dataset suitable for exploratory data analysis and visualization.

### 2. Assignment 7.2 – Activity 5.01: Clustering Sales Data Using K-Means
- **File:** `Assignment 7.2.ipynb`  
- **Description:** This notebook demonstrates K-Means clustering on a sales dataset.  
- **Key Steps:**
  - Data cleaning and normalization
  - Exploratory Data Analysis (EDA)
  - Application of K-Means clustering
  - Interpretation of clusters
- **Outcome:** Identification of meaningful customer or sales segments using unsupervised learning.

## Technologies Used
- Python 3.x  
- Jupyter Notebook  
- pandas, numpy, matplotlib, seaborn, scikit-learn
